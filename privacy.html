Privacy Policy

Service: AI Playlist Generator  |  Effective Date: January 1, 2026  |  Last Updated: March 20, 2026

1. Introduction

This Privacy Policy describes how AI Playlist Generator ("the Service," "we," "us," or "our") handles information when you use our web-based application. The Service enables users to generate music playlists using Google Gemini AI and create YouTube playlists via the YouTube Data API v3.

By using the Service, you acknowledge that you have read and understood this Privacy Policy.

2. Information We Access

The Service accesses the following information solely to provide its core functionality:

  • Gemini API Key: Provided by the user and used exclusively to make requests to the Google Gemini API for playlist generation. This key is never transmitted to our servers.
  • Google OAuth Client ID: Provided by the user and used solely to initiate the OAuth 2.0 authorization flow for accessing the YouTube Data API.
  • Google OAuth Access Token: A temporary token issued by Google after user authorization, used solely to create YouTube playlists on behalf of the user. This token is held in browser memory only and is never persisted or transmitted to third parties.
  • Song Input Data: Text entered by the user (song titles and artist names) is sent directly to the Gemini API to generate playlist recommendations and to YouTube Data API for video search. This data is not stored or logged by the Service.

3. Information We Do NOT Collect

The Service does NOT collect, store, transmit, or share any user data on any server operated by us.

Specifically, we do not collect:

  • Personally identifiable information (name, email address, etc.)
  • API Keys or OAuth credentials
  • Google account information or profile data
  • Usage analytics or behavioral data
  • IP addresses or device identifiers
  • Cookies or local storage data

All processing occurs entirely within the user's browser. No backend server infrastructure is operated by this Service.

4. Use of Google APIs

The Service uses the following Google APIs and is subject to the corresponding Google policies:

  • Google Gemini API — for AI-based playlist recommendations. Subject to Google Gemini API Terms of Service.
  • YouTube Data API v3 — for searching videos and creating YouTube playlists. Subject to YouTube API Services Terms of Service and YouTube Terms of Service.
  • Google OAuth 2.0 (Google Identity Services) — for user authentication. Subject to Google Privacy Policy.

The Service's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.

5. Data Retention

The Service does not retain any user data. All information (API keys, access tokens, song inputs) exists only in the user's browser memory during an active session. Upon closing or refreshing the browser tab, all such information is permanently discarded.

6. Third-Party Services

By using this Service, you interact directly with third-party services operated by Google. We are not responsible for the privacy practices of these third-party services. We encourage you to review Google's Privacy Policy at https://policies.google.com/privacy.

7. Security

Because all operations are performed client-side within your browser, API keys and tokens are not exposed to our servers. However, users are solely responsible for the security of their API keys and OAuth credentials. We strongly recommend:

  • Never sharing your API Key or Client ID with others.
  • Revoking access tokens via Google Account Permissions after use.
  • Restricting the scope and usage of your Google Cloud credentials in the GCP Console.

8. Children's Privacy

The Service is not directed to individuals under the age of 13. We do not knowingly collect personal information from children. If you believe a child has used the Service in a way that violates this policy, please contact us.

9. Changes to This Privacy Policy

We reserve the right to update this Privacy Policy at any time. Changes will be reflected by updating the "Last Updated" date at the top of this page. Your continued use of the Service after any changes constitutes your acceptance of the revised Privacy Policy.

10. Contact

If you have any questions or concerns regarding this Privacy Policy, please contact us via the project's GitHub Issues page.
